# Wave Cleanup – AI-Powered Ocean Waste Detection Platform

This project simulates an AI-powered system for detecting ocean waste and visualizing pollution hotspots on an interactive map.

## Features
- AI-based waste detection (simulated)
- Interactive map visualization
- Backend API for hotspot generation

## Tech Stack
- Python (Flask)
- JavaScript
- Leaflet Maps

## How to Run

### Backend
cd backend  
pip install -r requirements.txt  
python app.py  

### Frontend
Open frontend/index.html in browser
